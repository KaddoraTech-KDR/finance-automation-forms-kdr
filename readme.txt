🎀Finance Automation Forms kdr🎀
Modern WordPress plugin to create finance forms (Invoice, GST, Payment, Quotation, Expense) with submissions management and automation-ready workflow.

🚀 Overview
Finance Automation Forms kdr helps businesses collect financial data and automate finance workflows directly from their WordPress website.
It provides ready-to-use finance forms, secure submissions storage, admin dashboards, and structured data for automation such as PDF generation, email sending, and exports.

Designed for agencies, accountants, consultants, and service businesses, the plugin simplifies invoice collection, GST registration, quotations, payments, and expense tracking — all in one system.

✨ Features
6 built-in finance forms

1. Invoice
2. Payment
3. Quotation
4. Billing
5. Expense
6. GST Registration

Modern responsive UI
Admin submissions dashboard
Customer & form data viewer
File uploads support
CSV export
Shortcode embedding
Secure data storage
Automation-ready architecture (PDF / Email / Export / Payment)
